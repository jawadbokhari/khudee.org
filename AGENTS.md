# AGENTS.md

This file provides guidance to any AI coding tool (Claude Code, Codex, Gemini CLI, etc.) when working with content in this repository.

## Overview

This directory contains all materials for **Khudee.org** — a community for software delivery teams in Pakistan focused on developing critical thinking, systems thinking, and emotional intelligence before delegating execution to AI.

There is no software project here. Content is Markdown-based documentation and community assets.

## Key File

- `khudi-ai-community-brief.md` — The foundational community brief: name/meaning, audience, problem statement, methodology (Define → Analyze → Validate → Orchestrate), and founder context. (Note: predates the rename to Khudee.org)

## Context

- **Founder:** Jawad Bokhari, Product Manager, Lahore, Pakistan
- **Domain:** khudee.org (registered)
- **Philosophy:** Rooted in Allama Iqbal's concept of *Khudi* (خودی) — self-realisation and elevation of human potential
- **Mission:** Help Pakistani software teams develop upstream thinking skills before using AI for execution
- **Related platforms:** lovagility.com, jawadbokhari.substack.com

## Working in this Directory

When helping with content in this directory, preserve the voice and philosophy of the brief — measured, human-centred, grounded in Iqbal's thought. New materials should align with the four practices (Define, Analyze, Validate, Orchestrate) and three mindsets (Critical Thinking, Systems Thinking, Emotional Intelligence).
